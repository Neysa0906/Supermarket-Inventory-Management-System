# Supermarket Inventory Management System

This project is a web-based inventory management system developed to help supermarkets efficiently manage products, stock levels, and inventory operations.

The system automates inventory-related tasks such as adding products, updating stock, selling items, deleting records, and searching for products. It replaces traditional manual inventory tracking with a centralized digital solution.

## Project Overview

Managing inventory manually can lead to stock inaccuracies, delays, and human errors. This project provides an automated solution that maintains real-time inventory records and improves operational efficiency.

The system uses a MySQL database to store product information and PHP for backend processing, while HTML and CSS provide a simple and user-friendly interface.
